The Patient viewer application It is targeted at the Clinician or Information Analyst, and will display the resources from a patient record in a number of perspectives. These include:
  • The resource explorer to examine individual resources
  • The resource references graph that shows connections between resources
  • The terminology explorer that displays coded data
  • Numeric Observations - tables and graphs
  • The encounter timeline that shows resources by encounter
  • A FhirPath tab for querying within individual resources or the bundle of all resources
To select a Patient to view, first select the server where the data is stored using the link at the top left (Labelled 'Server'). https://hapi.fhir.org/baseR4/ is a good choice if you don't have a specific server in mind.
The icon will display a dialog where you can select another one, or enter the details of some other Server.

Next, click the 'Select Patient' button at the top right to select the patient to view from that server. You can choose the patient by their name, identifier or Id - or you can create a new one directly. Be aware that it might take a few seconds for a search to be performed.

As an alternative, you can select a sample patient from the list below. This is often the easiest if you don't have a specific server/patient in mind (and it's significantly faster to load!).

Some caveats:
  • You shouldn't use this for real patient data - there's no security
  • It's a training tool and will be really slow with large datasets (> 100 resources or so)
